### What I learned

1. There are 12 columns in Boostrap and if you want to use the col class, you must have the row class included.
```
<div class="row align-items-start">
    <div class="col">
      One of three columns
    </div>
    <div class="col">
      One of three columns
    </div>
    <div class="col">
      One of three columns
    </div>
</div>
```
2. Breakpoints in bootstrap
```
$grid-breakpoints: (
  xs: 0,
  sm: 576px, // >= 56px
  md: 768px, // >= 768px
  lg: 992px, // >= 992px
  xl: 1200px, // >=1200px
  xxl: 1400px // >= 1400px
);
```
